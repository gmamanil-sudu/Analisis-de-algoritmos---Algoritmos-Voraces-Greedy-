# Algoritmos Greedy en Java

## Descripción
Este proyecto contiene la solución de dos problemas utilizando la estrategia **Greedy (Voraz)** en Java. En ambos casos se busca obtener una solución eficiente tomando la mejor decisión posible en cada paso.

Los problemas desarrollados son:

- Mochila Fraccional
- Cobertura de Antenas WiFi
  
## Mochila Fraccional
**Objetivo:** Maximizar el valor total de los objetos sin exceder la capacidad de la mochila.

**Estrategia:** Seleccionar primero los objetos con mayor relación valor/peso y, si es necesario, tomar una fracción del último objeto.

**Complejidad:** O(n log n)

## Cobertura de Antenas WiFi
**Objetivo:** Cubrir todas las casas utilizando la menor cantidad posible de antenas.

**Estrategia:** Colocar cada antena en la posición que permita cubrir la mayor cantidad de casas cercanas.

**Complejidad:** O(n)

## Tecnologías utilizadas
- Java
- Algoritmos Greedy
